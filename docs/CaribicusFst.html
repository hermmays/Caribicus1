CaribicusFst

Fixed differences, FST, and Nei’s distance

Goals

What follows is our workflow for estimating pairwise fixed differences, genetic differentiation (FST), and Nei’s genetic distance among galliwasp species and populations based on the filtered SNP dataset generated from genotyping-by-sequencing (GBS) data.

Getting started

We are using R version (v) 4.5.2 and RStudio v 2026.01.0-392 for the analysis. Quarto v 1.6.39 for MacOS was used to document the workflow. Set the working directory in RStudio to one where the filtered vcf.gz file and the taxon/population designation file are available.

General methods summary

Pairwise fixed differences were estimated using the gl.fixed.diff function in dartR.base v 0.98 (Gruber et al. 2018) under default parameters (alpha = 0.05, delta = 0.02) with p-values estimated for each pairwise comparison. Pairwise FST was estimated using the stamppFst function and pairwise Nei’s distances were calculated using the stamppNeisD function in StAMPP v 1.6.3 (Pembleton, Cogan, and Forster 2013). Both FST and Nei’s distance are displayed as heatmaps with groups ordered from the lowest to highest mean pairwise values using ggplot2 v 3.5.2 (ggplot2?). Groups for all analyses are defined by the pop column in Caribicus_pops1.txt.

Convert data to genlight format

Convert the VCF object to a genlight object, set ploidy to 2 for these diploid organisms, and assign populations from Caribicus_pops1.txt. The match() function ensures population assignments are ordered correctly relative to the genlight individual order regardless of sample ordering in the population file.

CaribicusGEN <- vcfR2genlight(CaribicusVCF)
ploidy(CaribicusGEN) <- 2
pop(CaribicusGEN) <- CaribicusPOPS$pop[match(indNames(CaribicusGEN), CaribicusPOPS$id)]
